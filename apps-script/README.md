# Han Taekwondo Korea Apps Script Backend

Create a Google Spreadsheet with sheets:
- Students
- Parents
- Instructors
- Classes
- Attendance
- Payments
- Promotions

Paste Code.gs into Google Apps Script.
Deploy as a Web App.

Endpoints:
?resource=students
?resource=parents
?resource=instructors
?resource=classes
?resource=attendance
?resource=payments
?resource=promotions
